# DESIGN_SYSTEM.md

# Philmont Trek 12-15 Field Guide
## Design System

> Version 1.0

---

# Design Philosophy

The guide should feel like a blend of:

- National Park Service trail guide
- USGS field notebook
- Scout handbook
- National Geographic field guide

Primary goals:

- Beautiful
- Educational
- Easy to navigate
- Print-friendly
- Consistent across every page

---

# Page Formats

## Collector Edition
- Size: 11 x 17 inches (Landscape)
- Resolution: 300 DPI
- Export: Press-quality PDF

## Field Edition
- Size: 5.5 x 8.5 inches (Portrait)
- Spiral bound
- Lightweight

---

# Grid System

- 12-column layout
- 0.5 in margins
- 0.2 in gutters
- 8 pt spacing system

---

# Typography

## H1
- 32–40 pt
- Bold
- Forest Green

## H2
- 20–24 pt
- Bold

## H3
- 14–18 pt
- Semi-bold

## Body
- 10–12 pt
- Serif font
- 1.4 line spacing

## Captions
- 8–9 pt
- Sans-serif

# Design Principles

1. One idea per section.
2. Teach before you describe.
3. Favor diagrams over dense text.
4. Keep every page visually consistent.
5. Leave room for note-taking.
6. Every page should inspire curiosity.
