# PHOTO_GUIDE

## Daily Photo Requirements
1. Hero landscape
2. Route map
3. Elevation profile
4. Geologic feature
5. Historic image (if available)
6. Plant
7. Wildlife
8. Camp/program
9. Night sky

## Optional
- Drone imagery
- Macro geology
- Historic vs modern comparison
- Panoramas

## Metadata
Record location, direction, photographer, date, caption, source, copyright.
